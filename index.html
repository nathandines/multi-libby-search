Multi Libby Search

What is this? How do I use it?
How do I find and use my library identifiers?

What is this?

Libby only allows you to search a single active library at a time. It can show availability of books at your other libraries, but only if your active library has the same book in the first place.

This tool allows you to specify all of the libraries which you'd like to search and perform a search of them simultaneously.

How do I use it?

  1. Type your search query into the "Search Query" field.
  2. Populate your list of libraries in the "Library Identifiers" text box.
  3. Click "Submit" and your search will be submitted to all of your libraries across multiple browser tabs.

How to find and use a library identifier from Libby
  1. Navigate to Libby in your browser and sign-in.
  2. Once you're signed in, select a library for which you need the identifier.
  3. On the homepage for the active library, the library identifier will be visible in the location bar (see the below example for the "City of Melbourne" library identifier, which is "melbournelib").
  4. Add the library identifier to its own line in the "Library Identifiers" field on the Multi Libby Search page.
  5. Repeat for each library which you would like to search. These will be saved in your browser for subsequent visits.
